Changelog - Änderungsprotokoll von julius.latina2-halle.de

Dieses Änderungsprotokoll gibt Ihnen einen Überblick über die verschiedenen Versionen der Seite und deren Buildnummer sowie das Veröffentlichungsdatum. Aus technischen Gründen können Sie auf keine älteren Versionen dieser Webseite zugreifen.

Hinweise und Bemerkungen zum Änderungsprotokoll:

Diese Webseite unterstützt den Internet Explorer seit Version 3.02.04 (Build 3.204) nicht mehr. Daher werden Inhalte möglicherweise nicht korrekt bzw. gar nicht dargestellt. Der Kompatibilitätsmodus schafft zwar teilweise Abhilfe, Sie sollten trotzdem zu einem aktuellen Browser wechseln, wenn Sie noch den Internet Explorer verwenden.

  • Verwendung eines CSS-Skript zum effizienteren Designen der Webseiten seit Version 4.00.07 (Build 4.007)
  • Verwendung eines auf PHP-basierenden Kontaktformulars seit Version 4.01.02 (Build 4.102)
  • Verwendung einer SSL-Verschülsselung seit Version 5.00.01 (Build 5.001)
  • Verwendung von Tracking- und Analysetools seit Version 5.02.06 (Build 5.206)
  • hauptversionsübergreifender Quelltext seit Version 6.00.01 (Build 6.001)
  • IP-Prüfverfahren und CAPTCHA-Protokolle zum Schutz vor Spam und automatisierten Vorgängen seit Version 8.03.07 (Build 8.307)
  • automatische Überprüfung des Browsers und Kompatibilitätsmodus für den Internet Explorer seit Version 9.02.04 (Build 9.204)
  • automatischer Erkennung und Optimierung für mobile Endgeräte seit Version 10.01.03 (Build 10.103)
